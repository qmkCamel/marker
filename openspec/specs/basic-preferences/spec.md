# basic-preferences Specification

## Purpose
定义本地基础偏好与低频管理入口，保证设置视图承接周起始日、统计窗口和归档管理等能力。
## Requirements
### Requirement: 用户必须能够管理本地基础偏好
系统必须（MUST）允许用户管理影响本地产品体验的基础偏好，例如周起始日和默认统计窗口。

#### Scenario: 用户修改周起始日偏好
- **WHEN** 用户在设置中将周起始日改为其他星期
- **THEN** 新偏好会被持久化
- **AND** 后续与周相关的配额和统计解释会读取该偏好

### Requirement: 设置视图必须提供低频管理入口
系统必须（MUST）在设置视图中提供归档追踪项入口和基础应用信息，而不是把这些低频操作放入 Today 主流程。

#### Scenario: 用户进入设置页
- **WHEN** 用户打开设置视图
- **THEN** 视图提供基础偏好项、归档追踪项入口和应用信息展示
- **AND** 这些内容不会干扰 Today 的高频记录流程
