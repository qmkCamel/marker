# Marker 品牌调性

## 核心定义

`Marker` 是一个面向健康与自我照料场景的记录与追踪产品。

它的核心不是督促用户“完成任务”，而是帮助用户以稳定、可信、不过度施压的方式，记录那些与照顾自己有关的持续行为与状态，例如习惯、用药、经期、身体反馈与日常节律。

一句话总结：

`Marker` 用温和、克制、专业的方式，帮助用户记录与照顾自己，并用明确但不过度刺激的反馈，确认每一次完成与变化。

## 品牌定位

- `产品角色`：个人健康 / 自我照料记录工具
- `主要价值`：帮助用户看见、维持并回顾自己对身体与生活节律的照顾
- `核心心智`：不是“我要完成更多”，而是“我在认真照顾自己”

`Marker` 不是下面这些产品：

- 不是待办清单或目标冲刺工具
- 不是靠强激励维持连续性的打卡产品
- 不是冷冰冰的医院系统或病历后台
- 不是过度可爱化、弱化可信度的生活方式玩具

## 调性方向

`Marker` 的调性应落在下面这个组合上：

- `温和`
- `专业`
- `可信`
- `克制`
- `被确认`

建议的权重是：

- `健康 / 自我照料专业感` 高于 `轻松玩乐感`
- `稳定可信` 高于 `强刺激激励`
- `温和陪伴` 高于 `管理和驱动`

可以把它理解成：

`Marker` 有健康产品的可信与清晰，但保留个人产品应有的温度与日常陪伴感。

## 用户关系

`Marker` 与用户的关系，不应该像教练、老板或评判者。

更合适的关系是：

- 一个安静但可靠的个人照料记录器
- 一个帮助用户保持节律感的陪伴工具
- 一个能认真保存微小完成与身体变化的私人空间

它对用户传达的信息应该更接近：

- “这次已经被记录下来”
- “你有在持续照顾自己”
- “这些变化值得被认真看见”

而不是：

- “你还没完成目标”
- “你今天落后了”
- “继续冲刺，不要中断”

## 语言与文案原则

`Marker` 的文案要体现 `专业但不生硬，温和但不幼态`。

建议遵循这些原则：

- `准确`：用词清楚，不模糊、不夸张
- `平静`：避免命令式和高压语气
- `确认`：记录后给出清晰反馈，让用户知道这件事被认真保存
- `克制`：不过度煽动情绪，不把每次记录包装成“胜利”

推荐使用的词：

- `记录`
- `保存`
- `确认`
- `照料`
- `状态`
- `节律`
- `回看`
- `已完成`

尽量少用或避免的词：

- `冲刺`
- `打卡挑战`
- `不许中断`
- `落后`
- `达标压力`
- `连胜`
- `任务清零`

推荐的文案风格示例：

- `今日已记录`
- `已保存`
- `今天的用药已确认`
- `最近 7 天的记录较稳定`
- `先记录下来，之后可以回看`

不推荐的文案风格示例：

- `快去完成今天的任务`
- `你已经落后 3 次`
- `继续保持连胜`
- `别中断`

## 专业感的来源

`Marker` 的专业感不应来自“医疗化腔调”，而应来自下面这些特征：

- 信息结构清楚
- 状态反馈准确
- 数据表达稳定
- 视觉层级明确
- 用词审慎，不做夸张承诺
- 让用户感到自己的记录是被认真对待的

也就是说，`Marker` 要有健康相关产品的可靠感，但不应看起来像诊疗系统。

它更像：

- `个人健康节律记录工具`
- `自我照料观察与追踪工具`

而不是：

- `医疗诊断系统`
- `医院信息化界面`

## 视觉表达建议

如果未来把这套调性延伸到视觉与交互，建议遵循以下方向：

- 使用低刺激、偏柔和但有秩序感的色彩
- 保持留白、整洁和清楚的层级
- 强调“标记完成”和“记录存在”的确认感，而不是炫耀式反馈
- 图形语言更偏 `标记`、`留痕`、`节律`、`稳定`，而不是游戏化奖励
- 适合使用克制的圆角与柔和组件，但不要过度可爱化

适合的视觉关键词：

- `柔和`
- `安静`
- `整洁`
- `可信`
- `有呼吸感`

不适合的视觉倾向：

- 过于荧光和高刺激的激励色
- 过重的医院蓝白模板感
- 过于卡通或玩具化的表现
- 把每次记录都做成夸张成就反馈

## 体验决策原则

当后续产品或设计方案出现分歧时，可以用下面这些原则判断是否符合 `Marker`：

1. 这是否帮助用户更好地照顾自己，而不是只是推动完成更多？
2. 这是否让用户感觉被确认，而不是被催促？
3. 这是否增强了可信度，而不是让产品显得轻浮或娱乐化？
4. 这是否保持温和和克制，而不是制造额外压力？

如果一个方案同时满足 `更清楚`、`更可信`、`更温和`，通常就更接近 `Marker` 的方向。

## 品牌短句建议

可选的品牌短句中，优先推荐下面这句：

`温和记录，认真照料。`

如果需要更偏说明性的版本，也可以使用：

`记录照顾自己的每一步。`

## 最终结论

`Marker` 应被定义为一个 `偏健康 / 自我照料专业感` 的产品，而不是泛任务工具。

它的理想气质是：

`温和、专业、可信、克制`

它给用户的感受应该是：

`我不是在被管理，我是在被认真地帮助记录和照顾自己。`
